# 🌍 Planet Merge

円形フィールドの中心にある太陽に向かって天体を投入し、同じ天体を3つ合体させて大きな惑星へと進化させるパズルゲームです。

**▶ Play: https://hikaru8249.github.io/requirements/**

---

## ゲームルール

1. 画面外周のどこかをクリック（タップ）して天体を投入する
2. 投入した天体は中心の太陽に引き寄せられ、コアの周りに積み重なる
3. **同じ種類の天体が3つ以上接触**すると自動で合体し、上位の天体になる
4. 天体がフィールドの外周に達するとゲームオーバー

---

## 天体の進化順

| 天体 | 得点 |
|------|------|
| 🪨 隕石 | 1 |
| 🌑 月 | 3 |
| 🔴 火星 | 9 |
| 🌍 地球 | 27 |
| 🪐 木星 | 81 |

木星同士が合体すると両方消滅してボーナス得点が入ります。

---

## 操作方法

| 操作 | 内容 |
|------|------|
| マウス移動 / スワイプ | 投入角度を変える |
| クリック / タップ | 天体を投入 |

### インジケータの見方

- 🟢 **緑の脈動ドット + 緑のライン** → 投入可能
- ❌ **赤い × マーク** → 前の天体が飛行中のため待機
- ✨ **緑の拡散リング** → 投入可能になった瞬間

## ローカルでの開発

```bash
npm install
npm run dev
```

`http://localhost:5173/` でゲームが起動します。

```bash
npm run build   # 本番ビルド
```
